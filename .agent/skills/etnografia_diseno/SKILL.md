# Etnografía para el Diseño e Innovación

Esta habilidad permite planificar y ejecutar investigaciones etnográficas "en el mundo real" (in the wild), centradas en entender el comportamiento humano para inspirar el diseño y la ingeniería.

## Cuándo usar esta habilidad
**SIEMPRE** utiliza esta habilidad cuando:
- Necesites entender cómo interactúan los usuarios con un producto o servicio en su entorno cotidiano.
- El usuario solicite un plan de observación de campo.
- Se trabaje en las fases iniciales de un proyecto de innovación (búsqueda de oportunidades).
- Quieras identificar necesidades "tácitas" o comportamientos no articulados por los usuarios.

## 1. El Marco de Trabajo AEIOU
Utiliza este marco para organizar observaciones y categorizar datos de campo:

- **A - Activities (Actividades):** Acciones orientadas a metas. ¿Qué intentan lograr las personas? ¿Cuáles son los pasos?
- **E - Environments (Entornos):** El arena completa donde ocurren las actividades. Espacios, iluminación, ruido, atmósfera.
- **I - Interactions (Interacciones):** Entre personas o entre una persona y un objeto. Son los bloques de construcción de las actividades.
- **O - Objects (Objetos):** Elementos del entorno, a menudo usados de formas imprevistas o creativas (workarounds).
- **U - Users (Usuarios):** Las personas que proveen el comportamiento. Preferencias, necesidades, roles y perfiles.

## 2. Métodos de Observación

### Video Etnografía
- **Cámaras Estacionarias:** Para monitoreo continuo de un espacio (pasillos, tiendas, oficinas) para ver patrones de flujo.
- **Cámaras Móviles (Handicams):** Para seguir a un usuario (shadowing) o realizar "tours" de escritorio/espacio.
- **Valor:** Permite revisar instancias infinitamente y compartir la "mirada" del investigador con el equipo de diseño.

### Observación Participante
- **Inmersión:** Vivir o estar presente en el contexto del usuario por periodos prolongados.
- **Jotting:** Tomar notas rápidas de campo para luego expandirlas en diarios de campo detallados.
- **Ethnomethodology:** Enfocarse en el orden microscópico de la interacción momento a momento.

## 3. Principios de Operación
1. **Focus on Behavior:** Lo que la gente *hace*, no solo lo que *dice* (discrepancias entre dicho y hecho).
2. **Make Work Visible:** Identificar los "secretos mejor guardados" de cómo se realiza realmente una tarea.
3. **In the Wild:** Realizar la investigación en el contexto natural, nunca en laboratorio.
4. **Collaboration:** Incluir a diseñadores e ingenieros en la observación para facilitar la transferencia de conocimiento.

## 4. Estructura de Salida Recomendada (Plan de Etnografía)
- **Objetivo:** ¿Qué queremos descubrir?
- **Sitios:** ¿Dónde observaremos?
- **Protocolo AEIOU:** Guía de qué mirar en cada categoría.
- **Estrategia de Registro:** Video, fotos, notas, entrevistas interceptadas.
